1       What does this application do?

 

This application demonstrates the following:

-   Some form of input (widget: textbox, radio button, checkbox, ...)

-   Some operation on the ui input in sever.R

-   Some reactive output displayed as a result of server calculations

-   Documentation so that a novice user could use your application.

-   The documentation should be at the Shiny website itself. Not an external link.

 

2       How do I test the features?

2.1      Some form of input

There is a text input box and check boxes on the left panel of the app to accept inputs. Try changing the Caption and notice that the Caption changes on the Main panel (IRIS tab) without having to press the GO button.

2.2      Some operation on the UI

You can select the columns from the IRIS data that you want to see in the table in the IRIS tab by selecting/un-selecting the check boxes on the left panel. But, notice that the changes do not reflect on the right panel without clicking on the GO button.

Try sorting the columns, using filters on each of the rows, using the SEARCH field etc.

2.3      Some reactive output

As explained in #2.2 the same check boxes can be used to manipulate the summary shown in the SUMMARY tab in the main panel but notice that there is no need to press the GO button this time.

This is done to show case different ways in which the same input can be used to affect data: either reactive or non-reactive.

2.4      Documentation

Click on the HELP tab to see the documentation that will help you use the application.

2.5      Help is not an external link.

On clicking the HELP tab notice that there is no redirect to any external link.
